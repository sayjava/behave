## Quick Start

```shell
npx behavior-server
```

This will start the sever on port 8080

| Configuration | Default  |                         Description |
| ------------- | :------: | ----------------------------------: |
| port          |   8080   | The port the sever should listen on |
| debug         |   info   |                                \$12 |
| zebra stripes | are neat |                                 \$1 |
